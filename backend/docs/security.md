# Безопасность и контроль доступа

## Роли
- engineer — создание и изменение дефектов
- manager — назначение ответственных, изменение статуса, отчёты
- customer — только просмотр

## Механизмы защиты
- JWT авторизация
- Rate limiting (100 запросов / мин)
- Helmet для защиты заголовков
- CORS с белым списком доменов
- Логирование всех запросов (morgan)
- События безопасности (events.log)
